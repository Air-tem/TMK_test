1. Версия Delphi - 10.3 
2. Для работы с базами данных используются компоненты FireDAC(стандартные для Delphi c версии 10.1); 
3. Сервер баз данных - MySQL. Все настройки подключения прописаны в компоненте - FDConnection1; 
4. Для корректной работы указать драйвера подключения к базе данных при первом запуске указать путь к библиотеке libmysql.dll(лежит в корне проекта); 
5. Если будете тестировать на своей БД, то достаточно перенастроить компонет FDConnection1. SQL запросы использованные в проекте +- стандартны и должны работать на любой БД
